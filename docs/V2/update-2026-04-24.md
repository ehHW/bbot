# 前端更新说明（2026-04-24）

本次更新聚焦系统公告、系统维护联动、聊天发送失败体验和上传边界保护。

## 主要更新

- 新增系统公告前端能力：
  - 新增头部公告组件 [src/Layout/Components/HeaderSystemAnnouncement.vue](src/Layout/Components/HeaderSystemAnnouncement.vue)
  - 新增系统设置与公告 API 封装 [src/api/system.ts](src/api/system.ts)
  - 新增系统状态 Store [src/stores/system.ts](src/stores/system.ts)
  - 新增系统类型定义 [src/types/system.ts](src/types/system.ts)

- 系统设置页面升级：
  - 支持维护开关与维护时间配置
  - 支持公告发布（含内容长度上限）
  - 公告输入区禁止拖拽缩放
  - 文件: [src/views/Settings/index.vue](src/views/Settings/index.vue)

- 聊天设置拆分：
  - 新增聊天通知设置页 [src/views/Chat/components/ChatNotificationSettingsWorkspace.vue](src/views/Chat/components/ChatNotificationSettingsWorkspace.vue)
  - 路由与设置入口同步调整 [src/router/routes.ts](src/router/routes.ts), [src/views/Chat/components/ChatSettingsPanel.vue](src/views/Chat/components/ChatSettingsPanel.vue)

- 聊天发送失败体验修正：
  - 直聊文本发送不再用“好友列表缺失”误判为“已不是好友”
  - 改为依据会话能力字段 `can_send_message`
  - 权限类失败文案不再展示在发送失败感叹号菜单中
  - 文件: [src/stores/chat/messageActions.ts](src/stores/chat/messageActions.ts), [src/stores/chat/messageFailure.ts](src/stores/chat/messageFailure.ts), [src/views/Chat/components/MessageWorkspace.vue](src/views/Chat/components/MessageWorkspace.vue)

- 上传限制增强：
  - 新增 1GB 文件上限常量并在上传入口前置拦截
  - 文件: [src/utils/fileUploader.ts](src/utils/fileUploader.ts), [src/stores/file.ts](src/stores/file.ts), [src/modules/chat-center/composables/useMessageWorkspaceComposerScene.ts](src/modules/chat-center/composables/useMessageWorkspaceComposerScene.ts)

- 请求异常兼容增强：
  - 404 HTML 响应转换为可读错误
  - 503 维护态触发登录态处理与提示
  - 文件: [src/utils/request.ts](src/utils/request.ts), [src/utils/error.ts](src/utils/error.ts)
